GUÍA DE INSTALACIÓN DE APP CREATIVASPRINT

Esta guía tratará tanto la instalación y uso de la aplicación en Android Studio como su APK desde celular

1. Requisitos:

Asegúrese de tener instalado:

Android Studio Koala o superior (fue utilizado Android StudioNarwhal).

JDK 17 (incluido en Android Studio).

Emulador Android o un dispositivo físico con Android 8 o superior.

Conexión a internet.

La app se conecta al backend generado en Xano, por lo que requiere conexión a internet para que funcione correctamente.


2. Clonar el proyecto desde GitHub en la terminal:

git clone https://github.com/SebastianFredesV/CreativasPrint.git

Abrir el proyecto con Android Studio:
Para esto, primero se abre Android Studio, luego en la barra superior se selecciona File → Open → (carpeta del proyecto recién clonado).

Por último, se debe ejecutar la app con la opción de la barra superior "Run", luego "Run 'app'" y estará corriendo la app.



3. PARA CELULAR COMO APK (la APK está sin firmar)

Transferir el archivo APK por cable USB.

Conecte el teléfono al PC mediante USB.

En el teléfono, seleccione Transferir archivos (MTP).

Copie el archivo creativasprint.apk dentro de cualquier carpeta del teléfono.

Abra el APK desde la carpeta donde fue guardado (generalmente será “Download”).

Toque Instalar.

Espere a que termine la instalación y toca Abrir.


4. Ingresar
Para ingresar con permisos de administrador se deben usar las siguientes credenciales:

Correo: admin@creativasprint.com
Contraseña: admin123

Y como usuario de prueba (o puede crear uno en el Registro):

Correo: cliente@creativasprint.com
Contraseña: cliente123


¡Listo!
La app ya está funcional y se puede navegar a través de ella.
